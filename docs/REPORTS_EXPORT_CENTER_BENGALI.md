# NEXO Ride Reports + Export Center

Admin Web App-এর **Reports** tab থেকে business monitoring করা যাবে।

## কী কী দেখা যাবে
- Total rides / completed rides
- Gross fare
- Platform commission
- Driver payout
- Driver payout pending
- Sub Admin pending commission
- Net platform commission
- Daily performance
- Top drivers
- Area-wise performance
- Sub Admin due/paid report

## Export
**Export Completed Rides CSV** চাপলে completed ride list CSV হিসেবে download হবে।

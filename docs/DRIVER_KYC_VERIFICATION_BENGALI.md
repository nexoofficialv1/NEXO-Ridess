# Driver KYC Verification - NEXO Ride

Driver submit করবে:
- Driver photo
- Toto vehicle photo
- Aadhaar number + Aadhaar photo
- Driving licence number + licence photo
- Toto vehicle number

Admin Web App > KYC tab থেকে:
- Document preview
- Verify & Approve
- Reject with reason

Verification-এর পর driver online হতে পারবে।

Production note: Aadhaar/licence sensitive data। Launch-এর আগে encrypted storage, access audit, retention rule এবং privacy policy update করতে হবে।

# NEXO Ride v2.0 Sprint-5A — Payment Wallet + Cancel + Ride Details

Server Update Only. APK rebuild not required.

## Added
- Passenger/Driver ride cancel option before ride starts.
- Paid ride cancellation creates refund review record.
- Ride Details modal on tapping any ride row.
- Details show pickup, drop, booking time, accepted/paid/start/complete/cancel timeline, fare, payment, commission, driver earning and settlement status.
- Driver wallet payout request button.
- Admin driver payout request monitor and pay action.

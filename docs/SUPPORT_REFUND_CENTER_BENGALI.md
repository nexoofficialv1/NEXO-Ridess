# NEXO Ride — Support + Refund Center

এই module-এ passenger/driver support ticket, complaint এবং refund request manage করা যাবে।

## Passenger/Driver
- Profile → Support / Complaint / Refund
- Ride link করে ticket করা যাবে
- Paid ride-এর refund request করা যাবে

## Main Admin
- Admin Web → Support tab
- Ticket In Progress / Resolve / Close
- Refund Review / Approve / Paid / Reject
- Refund reference / note save হবে

## Production Note
Real refund করার জন্য Razorpay refund API অথবা manual UPI refund reference ব্যবহার করতে হবে।

# NEXO Ride v1.0.32 V33 — Support + Refund Center

## Added
- Passenger/Driver Support Center
- Create support/complaint ticket with optional ride link
- Paid ride refund request flow
- Main Admin Support tab
- Admin ticket action: OPEN / IN_PROGRESS / RESOLVED / CLOSED
- Admin refund action: UNDER_REVIEW / APPROVED / REJECTED / PAID
- User/admin notifications for support and refund status updates
- Persistent `support_tickets` and `refund_requests` collections

## Important
- Refund flow is workflow-ready. Real refund payment requires Razorpay/manual payment reference in production.

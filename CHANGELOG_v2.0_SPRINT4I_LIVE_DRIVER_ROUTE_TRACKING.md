# NEXO Ride v2.0 Sprint-4I - Live Driver Route Tracking

Server Update Only.

## Added
- Passenger can see selected driver on live map after driver accepts.
- Shows driver-to-pickup distance and approximate ETA before ride start.
- After ride starts, shows driver-to-drop remaining distance and ETA.
- Full live ride map modal with refresh button.
- Driver app sends fresh GPS before accept/arrive/start/complete ride actions.
- Ride screen polling keeps driver GPS updated while open.

## No APK rebuild required
Deploy on SmartASP and reload app with cache clear.

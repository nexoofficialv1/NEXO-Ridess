# NEXO Ride v1.0.14 V15 — Driver Earnings + Rating Flow

Functional update focused on post-ride business flow.

## Added
- Driver earning calculation after trip completion.
- Default platform commission: 10%.
- Driver wallet/earning history screen.
- Pending payout value for driver settlement.
- Passenger can rate driver after completed ride.
- Driver average rating updates automatically.
- Admin Payment Monitor with total fare, driver payout, commission and pending payout.

## Flow
Request → Accept → Pay → OTP → Pickup → Start → Complete → Earnings + Rating

Cache/version: 114v15

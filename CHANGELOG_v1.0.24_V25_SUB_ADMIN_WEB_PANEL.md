# NEXO Ride v1.0.24 V25 — Sub Admin Web Panel

## Added
- Separate Area Sub Admin Web App at `/subadmin/`.
- Sub Admin login panel for area managers.
- Sub Admin dashboard with managed users, drivers, completed rides and commission summary.
- Sub Admin can add Passenger and Driver accounts in their own area.
- Sub Admin can approve/reject/offline drivers under own area.
- Sub Admin can see own commission entries and payout history.
- Sub Admin Live Monitor for area drivers and active rides.
- Root page now links Passenger/Driver App, Main Admin Web App and Sub Admin Web App.

## Business Rule
- Sub Admin commission is calculated from platform commission, not from total ride fare.

# Admin Mobile Responsive Fix — Bengali Guide

এই update-এ Admin panel-এর mobile screen issue ঠিক করা হয়েছে।

## কী ঠিক হয়েছে

1. উপরের বড় header আর dashboard card ঢেকে রাখবে না।
2. নিচের bottom menu swipe করে দেখা যাবে।
3. মোবাইলে সব tab খোলার জন্য উপরে **Admin Section** selector দেওয়া হয়েছে।
4. ছোট screen-এ card, stats, input, button size adjust করা হয়েছে।
5. Android navigation bar-এর জন্য bottom safe padding দেওয়া হয়েছে।

## ব্যবহার

Admin login করার পরে উপরে **Admin Section** dropdown থেকে tab বেছে নিন।
নিচের button bar-টাও left-right swipe করা যাবে।

## যদি পুরনো design দেখা যায়

Chrome cache clear করুন বা incognito mode-এ খুলুন।

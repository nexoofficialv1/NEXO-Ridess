# NEXO Ride v1.0.6 V7 — Splash Toto Animation

## Added
- App startup splash animation.
- A toto moves from the left side to the right side before the app opens.
- After the animation, the login/language/session flow continues automatically.
- Astra Technologies dark neon brand style retained.

## Improved
- Consent checkbox is no longer pre-ticked; user must tick it manually.
- Version/cache updated to `106v7` to reduce old browser cache issues.

## Flow
App open → NEXO Ride splash → Toto left-to-right animation → Login/Language page or existing logged-in dashboard.

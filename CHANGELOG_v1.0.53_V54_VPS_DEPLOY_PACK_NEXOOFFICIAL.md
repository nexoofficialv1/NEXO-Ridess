# v1.0.53 V54 - VPS Deploy Pack for nexoofficial.in

Stable deployment pack for hosting NEXO Ride under the official domain using a subdomain:

- App: `https://ride.nexoofficial.in/app/`
- Admin: `https://ride.nexoofficial.in/app/admin.html`
- Sub Admin: `https://ride.nexoofficial.in/subadmin/`

## Added

- VPS deployment scripts
- PM2 ecosystem config
- Nginx reverse proxy config
- Production `.env` example
- Backup script
- Release update script
- VPS health check script
- Bengali deploy guide
- Version updated to `1.0.53-V54_VPS_DEPLOY_PACK_NEXOOFFICIAL`
- PWA cache updated to `153v54`

## Important

Root domain `nexoofficial.in` is not disturbed. NEXO Ride should use only `ride.nexoofficial.in`.
